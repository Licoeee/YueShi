# Welcome Screen 2.1 Revision Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 基于第一版反馈修正欢迎页，让版式、蛋糕、CTA 和 reveal 转场都贴近确认稿，并在 reveal 过程中露出真实主页预览内容。

**Architecture:** 保留 `pages/welcome` 作为应用首屏，但把 reveal 的目标内容抽成共享首屏组件，让欢迎页和 `pages/index` 使用同一份主页预览结构。把按钮展开、reveal 直径和点击门禁继续收口到纯 TypeScript 模块，页面文件只负责驱动状态与跳转。

**Tech Stack:** WeChat Mini Program, TypeScript, WXML, WXSS, TDesign Mini Program, Node.js test runner via `tsx`
